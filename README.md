# 🎮 Jutopia

> 설치 없이 브라우저에서 바로 즐기는 HTML5 캐주얼 게임 플랫폼

**▶ 바로 플레이: https://mindweaver74.github.io/Jutopia/**

<br>

## ✨ 소개

Jutopia는 스마트폰·태블릿·PC 어디서든 터치 한 번으로 플레이할 수 있는 무료 HTML5 게임 모음입니다.
회원가입도, 설치도, 로딩 대기도 없습니다. 출퇴근길·쉬는 시간·잠깐의 여유에 딱 맞는 짧고 중독성 있는 게임들로 구성되어 있습니다.

## 🛠️ 도구

## 🎯 특징

- **무료 & 무설치** — 링크만 열면 바로 플레이, 회원가입 없음
- **모바일 최적화** — 터치 조작 중심 설계, 세로 화면 지원
- **가볍고 빠름** — 각 게임은 순수 HTML5/JavaScript 단일 페이지로 즉시 로딩
- **꾸준한 업데이트** — 새로운 게임과 도구가 계속 추가됩니다

## 📁 이 저장소

이 저장소는 Jutopia의 **게임 허브(메인 페이지)** 소스입니다. 각 게임은 독립된 저장소로 관리됩니다.

```
index.html      게임 허브 메인 페이지 (게임 목록 · 도구 · 소개 · FAQ)
privacy.html    개인정보처리방침
```
